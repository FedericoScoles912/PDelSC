Un diario local, responsive y servido con Node.js

Esta portada concentra la experiencia completa del proyecto: una interfaz armada con Bootstrap 5, un servidor local construido con modulos nativos de Node.js y una organizacion modular para que el contenido del diario se mantenga legible, escalable y facil de mantener.

El objetivo principal fue eliminar la dependencia de media queries manuales para el layout y reemplazarla por la grilla responsive de Bootstrap. El resultado es una portada que se adapta correctamente a celulares, tablets y pantallas grandes sin duplicar logica visual.

Todo el contenido queda dentro de la misma aplicacion: el servidor interpreta la ruta, lee un archivo HTML local con `fs` y lo ensambla dentro del layout principal antes de enviarlo al navegador.

El layout usa `container`, `row`, `col`, espaciados utilitarios, navbar responsive y cards. La interfaz queda mas consistente y mucho mas facil de mantener.

Cada noticia se guarda como un archivo HTML local dentro de `pages/articles/`. El servidor usa `http`, `fs` y `url` para detectar la ruta solicitada, leer el archivo correcto y mostrarlo dentro del contenedor principal del diario.
